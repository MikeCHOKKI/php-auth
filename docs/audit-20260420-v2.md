## 🟡 Mineur (3 items)

### 1. Magic numbers non nommés

**Fichiers** :

- `src/Application/Service/AuthenticationService.php` (lignes 35-36)
- `src/Infrastructure/Security/Rs256JwtSigner.php` (ligne 62)

**Description** : Valeurs numériques sans constante nommée.

```php
// AuthenticationService.php
int $accessTokenTtl = 900,      // 15 minutes
int $refreshTokenTtl = 604800   // 7 jours
```

**Recommandation** : Définir des constantes de classe.

---

### 2. Code HTTP en dur

**Fichier** : `src/Infrastructure/Web/Middleware/JwtMiddleware.php` (ligne 66)

**Description** : Code HTTP 401 en dur au lieu d'utiliser des constantes.

```php
http_response_code(401);  // À remplacer par constante
```

**Recommandation** : Utiliser `Response::HTTP_UNAUTHORIZED` de PSR-7.

---

### 3. Tests incomplets

**Description** : Seuls 3 tests unitaires présents (Email, UserId, BcryptPasswordHasher).

**Impact** : Couverture de tests insuffisante pour un système critique d'authentification.

**Recommandation** : Ajouter tests d'intégration (repositories) et tests fonctionnels (endpoints API).
